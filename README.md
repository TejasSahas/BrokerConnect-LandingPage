# Broker Connect - Landing Page

A beautiful, modern landing page for Broker Connect - a platform that connects real estate brokers to close deals more efficiently.

## Features

- 🎯 **Smart Matching Algorithm** - AI-powered matching between requirements and listings
- 💼 **Organized Collaboration** - No more messy WhatsApp/Facebook groups
- 📍 **Location-Based Matching** - Find brokers in your area or across India
- 📊 **Match Percentage System** - See exactly how well requirements match listings
- 💰 **Make More Money** - Close more deals with the right connections
- 🇮🇳 **Built for Indian Brokers** - Designed for the Indian real estate market

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Modern, responsive design

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
